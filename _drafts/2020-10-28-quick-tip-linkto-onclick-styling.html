Let's say you have such a link:

= link_to "Log in with Github", user_github_omniauth_authorize_path

It can take a few seconds for it to load. For links that take long to load, you can add data: { disable_with: "Connecting..." } , so that it "feels" better when you press it.

= link_to "Log in with Github", user_github_omniauth_authorize_path, data: { disable_with: "Connecting..." }
Results in:
Looks sexier, doesn't it?
Of course, you can replace the word "Connecting" with anything you like ;)

Here is a live demo of how it works in Corsego SaaS
